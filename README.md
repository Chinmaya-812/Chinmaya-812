# 💫 About Me:
I am a Full Stack Software Developer with 2.3 years of product-based experience at IBM ISL, building scalable and accessible web applications. Experienced in front-end development using HTML, CSS, JavaScript, React, Redux, TypeScript, and microfrontend architectures, along with integrating REST APIs and writing robust tests with Jest and React Testing Library.<br><br>Along with front-end development, I also contribute to backend services using Node.js and Express.js, working with PostgreSQL and MongoDB in microservice-based architectures.
